# מערכת שרת–לקוח עם תבנית Observer


---


להלן המשימה שעליכם לבצע:


1. כתבו מחלקה המתארת **לקוח בצד השרת**.


2. שמרו **כל חיבור של לקוח** במערך/רשימה של לקוחות.


3. בעת חיבור ראשוני של לקוח לשרת:
- בקשו מהלקוח לשלוח:
- שם פרטי
- שם משפחה
- כתובת אימייל
- וכל שאר השדות שהגדרתם במחלקת הלקוח
- **רק לאחר** שהלקוח השלים את הרישום:
- הוסיפו אותו לרשימת הלקוחות
- ורק אז אפשרו לו לדבר עם השרת


4. במהלך הרישום יש לשאול:
   **"האם אתה מעוניין להצטרף למערכת ההתראות?"**


5. תשובת הלקוח:
- YES → הוספה לרשימת ה־Observers
- NO→ הלקוח נשאר רגיל
